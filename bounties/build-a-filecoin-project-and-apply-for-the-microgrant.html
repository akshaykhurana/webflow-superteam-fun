To date, the Filecoin project has focused primarily on research, specification and implementation of a decentralized storage network protocol. Throughout this process they’ve worked with incredible community members and contributors. Some are already building tools on the budding network in active development.

The Filecoin Dev Grant program was created to reward ongoing & existing contributions that add great value to the Filecoin ecosystem, inspire more contributors to solve open problems & seed the creation of new products, businesses and tools to increase the utility of Filecoin

Filecoin is now looking to fund talented and eager teams that want to work with them to build a more robust, efficient, and decentralized web.

MISSION

 Build a Solana project that leverages Filecoin and apply for the micro-grant.

Evaluation Criteria

Resources

Proposal templates can be found on our GitHub site. Generally, all projects must meet the following requirements:

  • All projects must be open-sourced.
  • All projects must be dual-licensed under MIT and APACHE2 licenses.
  • All teams must be self-managed.

Once you submit a proposal, a member of the Dev Grants team will review it on GitHub and either approve, reject, or request changes on your proposal. If your grant is approved, congratulations! We’ll connect with all grant winners to finalize legal and financial details, funding milestones, and timeframe.
